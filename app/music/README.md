# Thư mục Music

Đặt file nhạc nền của bạn vào thư mục này.

File nhạc sẽ được tự động copy vào `.tmp/music` và `dist/music` khi chạy gulp serve hoặc gulp build.

**Lưu ý:** 
- File nhạc hiện tại được tham chiếu trong code: `Em Đồng Ý (I Do).mp3`
- Đảm bảo tên file khớp với đường dẫn trong `app/scripts/main.js` (dòng 151)
- Hoặc cập nhật đường dẫn trong code nếu bạn đặt tên file khác
